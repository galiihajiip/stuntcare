## Pendahuluan

STUNTCARE adalah platform digital yang membantu Posyandu dan ibu balita dalam memantau pertumbuhan anak secara realtime menggunakan standar WHO. Platform ini membantu deteksi dini risiko stunting dan memberikan rekomendasi gizi yang tepat.

### Apa itu Stunting?

Stunting adalah kondisi gagal tumbuh pada anak akibat kekurangan gizi kronis, terutama dalam 1000 hari pertama kehidupan. Anak dengan stunting memiliki tinggi badan yang lebih pendek dari standar usianya.

## Memulai

### Registrasi Akun

1. Buka aplikasi STUNTCARE
2. Klik "Daftar" di halaman utama
3. Pilih peran Anda:
   - **Kader Posyandu**: Untuk petugas kesehatan
   - **Ibu/Wali**: Untuk orang tua
4. Isi formulir pendaftaran:
   - Nama lengkap
   - Email
   - Nomor HP (opsional)
   - Password (minimal 6 karakter)
5. Klik "Daftar"
6. Cek email untuk verifikasi (jika diperlukan)

### Login

1. Masukkan email dan password
2. Atau gunakan "Login dengan Google"
3. Klik "Masuk"

## Untuk Kader Posyandu

### 1. Membuat Komunitas

Setelah registrasi pertama kali:

1. Isi informasi posyandu:
   - Nama posyandu
   - Alamat lengkap
   - Provinsi, Kota, Kecamatan
   - Nomor kontak
2. Klik "Selesai"
3. Sistem akan generate kode undangan otomatis

### 2. Mengelola Komunitas

**Melihat Kode Undangan:**
1. Menu "Komunitas"
2. Lihat kode 8 karakter
3. Klik ikon copy untuk menyalin
4. Bagikan ke ibu-ibu di posyandu

**Melihat Anggota:**
- Daftar semua anggota komunitas
- Role masing-masing (Admin/Kader/Anggota)
- Tanggal bergabung

### 3. Menambah Data Anak

1. Menu "Daftar Anak" â†’ "Tambah Anak"
2. Isi data:
   - Nama lengkap anak
   - NIK (opsional)
   - Tanggal lahir
   - Jenis kelamin
   - Email orang tua (jika sudah terdaftar)
3. Klik "Simpan"

### 4. Input Pengukuran

1. Menu "Input Pengukuran"
2. Pilih anak dari dropdown
3. Isi data pengukuran:
   - Tanggal pengukuran
   - Tinggi badan (cm)
   - Berat badan (kg)
   - Lingkar kepala (opsional)
4. Klik "Simpan"

**Sistem akan otomatis:**
- Menghitung Z-Score (HAZ, WAZ, WHZ)
- Menentukan status stunting
- Update status anak

### 5. Melihat Detail Anak

1. Klik kartu anak di "Daftar Anak"
2. Lihat informasi:
   - Status stunting terkini
   - Pengukuran terakhir
   - Riwayat pertumbuhan
   - Data orang tua

### 6. Membuat Pengumuman

1. Menu "Pengumuman" â†’ "Buat Pengumuman"
2. Isi judul dan isi pengumuman
3. Klik "Publikasikan"
4. Semua anggota akan melihat pengumuman

### 7. Membuat Event

1. Menu "Event" â†’ "Buat Event"
2. Isi informasi:
   - Judul event
   - Deskripsi
   - Tanggal & waktu
   - Lokasi
3. Klik "Buat Event"

### 8. Feed Komunitas

1. Menu "Feed"
2. Tulis postingan di kolom input
3. Tekan Enter atau klik tombol kirim
4. Lihat postingan dari anggota lain
5. Berikan like atau komentar

### 9. Laporan

1. Menu "Laporan"
2. Lihat statistik:
   - Total anak terdaftar
   - Jumlah anak normal
   - Jumlah anak berisiko
   - Jumlah anak stunting
3. Klik "Export PDF" untuk unduh laporan

## Untuk Ibu/Wali

### 1. Bergabung dengan Komunitas

1. Menu "Gabung Komunitas"
2. Masukkan kode undangan (8 karakter)
3. Kode didapat dari kader posyandu
4. Klik "Bergabung"

### 2. Menambah Data Anak

1. Menu "Tambah Anak"
2. Isi data anak:
   - Nama lengkap
   - NIK (opsional)
   - Tanggal lahir
   - Jenis kelamin
3. Klik "Simpan"

### 3. Memantau Pertumbuhan

**Dashboard:**
- Lihat semua anak Anda
- Status stunting masing-masing
- Pengukuran terakhir

**Detail Anak:**
1. Klik kartu anak
2. Lihat:
   - Status stunting
   - Tinggi & berat badan terakhir
   - Riwayat pengukuran lengkap
   - Grafik pertumbuhan

### 4. Edukasi Kesehatan

1. Menu "Edukasi"
2. Pilih kategori:
   - Gizi
   - Sanitasi
   - Imunisasi
   - Pola Asuh
3. Klik artikel untuk membaca
4. Tandai sebagai selesai setelah membaca
5. Progress Anda akan tersimpan

### 5. Analisis Gizi

1. Menu "Analisis Gizi"
2. Pilih makanan yang dikonsumsi hari ini
3. Atau klik "Muat Menu Rekomendasi"
4. Klik "Analisis Nutrisi"
5. Lihat hasil:
   - Skor nutrisi (0-100)
   - Total kalori, protein, zat besi
   - Rekomendasi perbaikan

### 6. Cek Gejala (Symptom Checker)

## FAQ

### Umum

**Q: Apakah STUNTCARE gratis?**
A: Ya, STUNTCARE gratis untuk semua pengguna.

**Q: Apakah data saya aman?**
A: Ya, kami menggunakan enkripsi dan Row Level Security untuk melindungi data Anda.

**Q: Bisa diakses dari HP?**
A: Ya, STUNTCARE responsive dan bisa diakses dari HP, tablet, atau komputer.

### Untuk Kader

**Q: Bagaimana jika lupa kode undangan?**
A: Buka menu "Komunitas" untuk melihat kode undangan Anda.

**Q: Bisa menambah kader lain?**
A: Ya, bagikan kode undangan. Setelah bergabung, ubah role mereka di menu anggota.

**Q: Bagaimana cara menghapus data anak?**
A: Hubungi admin untuk menghapus data (fitur akan ditambahkan).

### Untuk Ibu

**Q: Anak saya belum terdaftar di posyandu, bisa pakai STUNTCARE?**
A: Ya, Anda bisa tambah data anak sendiri. Nanti bisa dihubungkan dengan posyandu.

**Q: Bagaimana cara melihat grafik pertumbuhan?**
A: Klik detail anak, grafik akan muncul jika sudah ada minimal 2 pengukuran.

**Q: Symptom checker akurat?**
A: Symptom checker adalah alat bantu, bukan pengganti dokter. Selalu konsultasi dengan tenaga kesehatan.

### Teknis
